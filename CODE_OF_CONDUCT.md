# Code of Conduct

## Our Pledge
We are committed to a respectful, professional, and technically focused
environment for everyone participating in OpenVisionMatrix.

This project exists to build a projection-mapping PWA and its supporting
math/domain packages. Discussions and contributions should reflect that goal.

## Expected Behavior
Participants are expected to:
- Be respectful and constructive
- Focus on technical merit and factual discussion
- Accept constructive feedback
- Respect differing viewpoints and experience levels
- Communicate clearly and professionally

## Unacceptable Behavior
Unacceptable behavior includes:
- Harassment, insults, or personal attacks
- Discriminatory or offensive language
- Trolling, deliberate disruption, or bad-faith participation
- Publishing private information without consent
- Attempts to pressure maintainers into decisions

## Scope
This Code of Conduct applies to:
- GitHub issues
- Pull requests
- Discussions
- Project-related communication channels

## Enforcement
Project maintainers are responsible for enforcing this Code of Conduct.
They may take appropriate action, including:
- Warning the offender
- Removing comments or contributions
- Temporarily or permanently banning participants

Decisions are made in the interest of project quality, stability, and
long-term maintainability.

## Reporting
If you experience or witness unacceptable behavior, please report it to:

Email: conduct@bunkerschild.de

All reports will be handled discreetly and reviewed fairly.
